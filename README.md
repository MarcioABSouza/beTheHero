# beTheHero
Aplicação para ONGs encontrarem pessoas dispostas a ajudarem nas suas mais diversas demandas. 

## Tecnologias utilizadas

Front - React, Axios. 

Back - Node Js, Express, SQLite3, Knex

Mobile - React Native, Expo
